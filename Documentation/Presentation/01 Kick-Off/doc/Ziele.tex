\section{Ziele}
\frame{\frametitle{Ziele}
Das Ziel ist eine Android App zu erstellen, welche für alle Medien in den Bereichen.
\begin{itemize}
	\item Bücher
	\item Filme
	\item Konsolenspiele
\end{itemize}
Eine schnelle und einfach zu handhabende Sammlungsverwaltung bereitstellt.\\[2ex]
Neue Medien sollen durch scannen mit der Handy Kamera hinzugefügt werden können.
}

\frame{
Die App unterstützt den Besitzer mit folgenden Funktionalitäten.
\begin{itemize}
	\item Anlegen eines neuen Items
	\item Ausgabe der Sammlung (inklusive Filterfunktion)
	\item Manuelles bearbeiten eines Items.
	\item Löschen eines Items.
	\item Exportieren der kompletten oder gefilterten Sammlung.
	\item Verwalten verliehener Items.
\end{itemize}
}
